1. Mục đích sử dụng: phân tích các bài báo xem nó thuộc chủ đề nào
2. Cách sử dụng: trước tiên, người dùng thực hiện chức năng 'huấn luyện' để chương trình bắt đầu học máy từ dữ liệu đã cho trước. Tiếp đến, bạn chọn file để phân tích, chương trình hộ trợ các file 'docx, txt' và chọn chức năng phân tích. Chương trình sẽ tiến hành phân tích xem bài báo bạn đưa bào thuộc loại chủ đề nào.
3.Người dùng có thể chỉnh sửa file huấn luyện ở 'Tệp huấn luyện'. Chương trình đã nhập sẵn các bài báo nằm trong các chủ để(label) "Thể thao,Pháp luật,Sức Khỏe,Giáo dục" được đánh các label lần lượt từ "0 -> 3". Người dùng có thể thay đổi theo nhu cầu của mình, có thể thêm chủ đề tiếp theo là 'Giải trí' với lable là '4',...

Sau khi chỉnh sửa, bạn chọn chức năng 'Bắt đầu huấn luyện'. Dữ liệu huấn luyện sẽ được cập nhập lại và bạn tiến hành sử dụng chương trình như bình thường.
